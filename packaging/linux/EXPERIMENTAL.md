# Experimental Linux build

This Linux artifact has not yet been exercised on a clean Linux host. It is a
best-effort preview of EMTG's Linux build and packaging support, not a
production-supported release target.

Please report the distribution and version, CPU architecture, compiler version,
build command, and complete error output when reporting a Linux-specific issue.
Windows x64 remains the validated release target until clean-host Linux package,
installation, runtime, and uninstall tests have passed.
